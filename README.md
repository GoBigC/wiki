# BigC Wikipedia

Requirements: 
- python3 
- mkdocs 

```pip install mkdocs```
- mkdocs-material 

```pip install mkdocs-material```

Steps to add docs: 
- Add files to `docs/`
- Reference your file in `mkdocs.yml`
- Commit the code to `master`
- Run 

```mkdocs gh-deploy``` 

to build and deploy automatically
