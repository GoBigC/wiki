# BigC Wikipedia

Requirements: 
- python3 
- mkdocs 
```pip install mkdocs```
- mkdocs-material 
```pip install mkdocs-material```

Steps to add docs: 
- Add files to `docs/`
- Reference your file in `mkdocs.yml`
- Build the site
```mkdocs build```
- Push everything to the `master` branch & wait for automatic deployment
